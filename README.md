# iPhone Sales Analysis 

## Project Overview

This project analyzes iPhone sales data using Python to uncover patterns, trends, and insights from the dataset. The analysis includes data cleaning, exploration, and visualization to better understand how different factors influence iPhone sales.

## Objective

The main objective of this project is to:

* Analyze iPhone sales data
* Identify trends and patterns
* Visualize insights using charts and graphs
* Demonstrate data analysis skills using Python

## Tools & Technologies Used

* Python
* Pandas
* NumPy
* Plotly / Matplotlib
* Jupyter Notebook

## Dataset

The dataset contains information related to iPhone sales including product details, ratings, prices, and other attributes used for analysis.

## Steps Performed

1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Insights Generation

## Key Insights

* Certain iPhone models have significantly higher ratings and sales.
* Price variations influence customer preferences.
* Visualizations help identify trends in product performance.

## Project Files

* `iphone_sales_analysis.ipynb` – Jupyter Notebook containing the full analysis
* `apple_products.csv` – Dataset used for the project

## Conclusion

This project demonstrates how Python can be used for data analysis and visualization to extract meaningful insights from sales data.
